# SF Pro Text – local web fonts

This folder contains SF Pro Text `.otf` files used by the site:

| File | Weight | Used for |
|------|--------|----------|
| `SF-Pro-Text-Regular.otf` | 400 | Body text, subtitles, descriptions |
| `SF-Pro-Text-Medium.otf` | 500 | Hero tagline (e.g. “Welcome to my portfolio”) |
| `SF-Pro-Text-Semibold.otf` | 600 | Gallery item titles |

The page loads these via `@font-face` in `index.html`. No conversion is needed—browsers support OpenType (`.otf`) directly.
