Project One

A robotics project combining sensing and actuation. This work explores how to make systems that adapt in real time and respond to changing environments with precision and reliability.

Project Two

Design and control of a custom robotic platform. From mechanical design to embedded software, this project demonstrates end-to-end development of a mobile system.

Project Three

Computer vision and perception for autonomous navigation. This project integrates cameras and depth sensors to build a robust understanding of the surrounding world.

Project Four

Manipulation and grasping with robotic arms. Focus on planning and execution of pick-and-place tasks in semi-structured environments.

Project Five

Multi-robot coordination and swarm behaviors. Experiments with communication, formation control, and emergent strategies for collaborative tasks.

Project Six

Human-robot interaction and safety. This project addresses how robots can work alongside people safely and intuitively, with clear feedback and predictable motion.
